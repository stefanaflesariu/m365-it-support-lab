# Project 2 – Password Reset & MFA

##  Teil 1 – Password Reset (Maria Meier)

### Schritte
1. Im Microsoft 365 Admin Center den Benutzer Maria Meier geöffnet  
2. Auf Kennwort zurücksetzen (Reset password) geklickt  
3. Neues temporäres Passwort erstellt  
4. Option aktiviert: „Der Benutzer muss das Kennwort bei der nächsten Anmeldung ändern“
5. Benutzer informiert und Test-Login durchgeführt  
6. Benutzer konnte sich erfolgreich anmelden und neues Passwort setzen

##  Teil 2 – Multi-Factor Authentication (Thomas Keller)

### Schritte
1. Im Admin Center unter Active Users → Multi-Factor Authentication den Benutzer Luca Steiner ausgewählt  
2. MFA-Status geändert auf Enabled
